# Project Description: Word Runner

**Date:** 2025-12-02T13:55:12+04:00

## Overview
"Word Runner" is a web-based infinite scroller game set in a rectilinear, New York-style cityscape. The player navigates an infinite city environment while interacting with words.

## Goals
- [ ] Create an infinite scrolling cityscape environment.
- [ ] Implement a character navigation system for rectilinear streets.
- [ ] Implement a scoring system based on collecting an equal balance of word types.

## Gameplay Mechanics
- **Objective:** Collect an equal amount of verbs, nouns, and adjectives.
- **Setting:** Infinite cityscape with rectilinear streets (grid-like).
- **Collection:** Collision-based. The character physically runs into words floating in their lane.
