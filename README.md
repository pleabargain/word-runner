# Word Runner

**Last Updated:** 2025-12-02T14:09:58+04:00

**Word Runner** is a web-based infinite scroller game set in a rectilinear, New York-style cityscape. The player navigates an infinite city environment while interacting with words.

## Gameplay
- **Objective:** Navigate through time frames (Past, Present, Future) and collect words.
- **Word Types:** Verbs, Nouns, Adjectives, Days of the Week, Months
- **Time Frames:** 
    - **Past** (Sepia colors): Collect past tense verbs (e.g., "Ran", "Jumped")
    - **Present** (Blue/Gray colors): Collect present tense verbs (e.g., "Run", "Jump")
    - **Future** (Cyan/Neon colors): Collect future tense verbs (e.g., "Will Run", "Will Jump")
- **Controls:** 
    - **Arrow Keys:** Move Left/Right
    - **Space Bar:** Pause/Resume Game
- **Difficulty:** Select from A1 (Beginner) to C2 (Proficiency) to adjust word complexity.
- **Mechanics:** Collide with words to collect them. Time frames change every 1000 distance units. Words appear in bright white in the center lanes.

## Development
This project is built using HTML, CSS, and Vanilla JavaScript.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
